建立个人博客

2018/2/15 posted in  说人生

参考资料

参考资料一：如何搭建个人独立博客？ - 陈素封的回答 - 知乎
参考资料二：一步步在GitHub上创建博客主页
初心：
输出自己整理自己知识
有一个发声的地儿
长尾不求流量
目标：
申请个人公众号：志鹏世说 以及 个人博客www.zhipeng.ink

配置 SSH keys

涉及到的工具

  • Mweb：博客书写工具
  • Github Desktop：桌面管理与上传
  • Git：Git环境配置

步骤

一、配置Git环境

二、网站GitHub的设置

  1. 设置一个仓库username.github.io
  2. 记录下红框部位的代码，红框右上角处图标可以复制到剪切板

三、设置Mweb，生成本地网站并预览

  1. 设置本地静态网站的文件夹，名字可以不和username.github.io一样

  2. 设置几个分类，尝试往里面放几个文章

  3. 到terminal里面运行刚才红框部分里面的代码

  4. 此步有错，就直接去github desktop里面运行。

四、更新方法

  1. 写文章
  2. 文件夹右键点击生成网站

  3. GitHub desktop里面更新

  • 实测大约要20-40分钟才能看到
  • 之后每次都右键点一下生成网站，然后在GitHub desktop里同步到项目中。

节约空间的方法

  1. 照片处理：先上传到七牛云，获取超链接。
